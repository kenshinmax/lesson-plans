WHAT YOU WILL NEED

What you will do

Over the course of this project work, you will start working with forms, use a combination of input types, add progress indicators and learn some techniques for form validation.

What you will learn

By making Forms with Oracle JET you will learn:

  • How to use different input components
  • How to arrange input components to fit any device
  • Validate single components and entire forms
  • Use converters to handle user data entry
  • How to incorporate progress indicators
